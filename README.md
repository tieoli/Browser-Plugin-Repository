# Edge 书签管理

一个轻量、直接、适合日常整理的 Edge/Chrome 书签管理扩展。它提供独立管理页面，支持文件夹树浏览、多选拖拽、批量移动、搜索、重复书签检测、空文件夹检测、删除确认和撤销。

## 功能亮点

- 书签树管理：左侧展示浏览器书签文件夹结构，支持展开、折叠和快速定位。
- 当前文件夹列表：右侧展示当前文件夹内的书签和子文件夹。
- 多选整理：支持勾选多个书签或文件夹后批量移动。
- 拖拽排序：在右侧列表中拖拽调整顺序，点击“同步排序”后写回浏览器书签。
- 拖拽移动：把书签拖入左侧目录或右侧子文件夹，快速归类。
- 搜索筛选：按标题、网址、域名搜索书签。
- 重复书签检测：按规范化 URL 查找重复书签，方便清理。
- 空文件夹检测：一键列出没有内容的文件夹。
- 安全删除：删除书签或文件夹前会弹出确认，避免误删。
- 撤销上一步：支持撤销移动、重命名、删除、新建文件夹和排序同步。
- 图标显示：书签显示站点 favicon，加载失败时使用默认图标。

## 安装方式

1. 下载或克隆本仓库。
2. 打开 Edge 或 Chrome 的扩展管理页面。
   - Edge: `edge://extensions/`
   - Chrome: `chrome://extensions/`
3. 开启“开发人员模式”。
4. 点击“加载解压缩的扩展”。
5. 选择本项目目录。
6. 点击浏览器工具栏中的扩展图标，即可打开书签管理页面。

## 使用说明

### 浏览书签

左侧是书签文件夹树，点击任意文件夹后，右侧会显示该文件夹内的书签和子文件夹。

### 新建文件夹

点击右上角“新建文件夹”，输入名称后会在当前文件夹下创建新文件夹。

### 重命名和删除

鼠标移动到文件夹或书签行上，会出现操作按钮：

- `✎` 重命名
- `🗑` 删除

删除前会弹出确认。误删后可以点击顶部“撤销”恢复。

### 多选移动

勾选右侧列表中的多个项目，然后拖动其中一个项目到目标文件夹，即可批量移动。

### 排序同步

在右侧列表中拖动项目调整顺序后，顶部“同步排序”按钮会变为可点击状态。点击后，排序会写回浏览器书签。

### 搜索

在顶部搜索框输入关键词，可按书签标题、网址和域名搜索。搜索结果支持删除和打开所在文件夹。

### 清理重复书签

点击“重复书签”后，会列出 URL 相同或规范化后相同的书签。可以勾选需要删除的项目，再点击“删除所选”。

### 清理空文件夹

点击“空文件夹”后，会列出没有任何子项的文件夹。可以逐个删除或批量删除。

## 权限说明

本扩展只申请以下权限：

- `bookmarks`：读取、创建、移动、更新、删除浏览器书签。
- `favicon`：显示书签对应网站的图标。

扩展不会上传、同步或发送任何书签数据到第三方服务器，所有操作都在本地浏览器内完成。

## 项目结构

```text
.
├── background.js    # 点击扩展图标时打开管理页面
├── manager.html     # 书签管理页面结构和样式
├── manager.js       # 书签读取、渲染、拖拽、搜索、清理和撤销逻辑
└── manifest.json    # Manifest V3 扩展配置
```

## 技术说明

- Manifest V3
- 原生 JavaScript
- Chrome/Edge `chrome.bookmarks` API
- 无构建步骤，无外部依赖

## 注意事项

- 删除文件夹会同时删除其中所有书签和子文件夹，请确认后再操作。
- 排序操作需要点击“同步排序”才会写回浏览器。
- 移动、删除、重命名、新建文件夹会立即写回浏览器书签，但可以通过“撤销”恢复最近操作。
